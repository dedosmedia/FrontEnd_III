**Ejercitación JSX**

Usando sintaxis JSX, crear un elemento h2 y un elemento ul que contenga un listado de elementos li, de manera que React produzca un código HTML similar al de abajo. Podés usar los textos que prefieras.
¡Mucha suerte!

 `<h2> Listado </h2>
    <ul>
    <li key={1}>soy</li>
    <li key={2}>una</li>
    <li key={3}>lista</li>
</ul>`
